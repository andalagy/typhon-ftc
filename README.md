## How to build & run

1. Open this project in Android Studio.
2. Make sure Android SDK / Platform and build-tools that match the FTC SDK version are installed.
3. Select the `FtcRobotController` module as the run configuration.
4. Plug in the Robot Controller phone via USB-A <-> USB-C cable.
5. Click Run â–¶ to install the app.
6. Connect the Driver Station phone and select `MyTeleOp` (or other OpModes) to test on the real robot.
